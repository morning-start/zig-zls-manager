## 性能优化建议（Phase 2.5）

1. **延迟加载**: 非必需模块（如 IDE 配置、ZLS API）可延迟初始化
2. **连接复用**: 启用 HTTP/2 和连接池，减少重复握手
3. **并发下载**: Zig + ZLS 使用 `tokio::join!` 并行下载
4. **缓存命中率**: 优化 API 缓存 TTL 策略
